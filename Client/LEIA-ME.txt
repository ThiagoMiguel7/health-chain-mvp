É necessário instalar o seguinte na maquina:


nvm install 20
nvm use 20
npm init -y
npm install @polkadot/api
npm i -D ts-node typescript @types/node

Depois disso pode rodar o Typescript para ver os resultados. Exemplos:


node --loader ts-node/esm helloWord.ts
npx ts-node frstConnectionTest.ts
node frstConnectionTest.js


Antes de executar todos esses comandos, recomendo iniciar o node substrate no endereço localhost. Porta default = 9944.

Antes de usar o KUBO (vide diretório kubo), rode nesse diretório o seguinte comando:

npm install ipfs-http-client
